# greentic-demo

Minimal Rust workspace for collecting many Greentic demos as independent crates.

## Layout

- `crates/<demo_name>/`: source for each demo crate.
- `demos/<demo_name>.gtbundle`: packaged demo bundles generated by the packaging script or GitHub Actions.
- `ci/local_check.sh`: local validation for the workspace and demo packaging flow.

## Add a Demo

1. Create a crate in `crates/<demo_name>`.
2. Add a standard `Cargo.toml` manifest for that demo.
3. Run `ci/local_check.sh` to verify the workspace and bundle output.

## Package Demos

Run:

```bash
scripts/package_demos.sh
```

Each crate directory under `crates/` is archived to `demos/<demo_name>.gtbundle`.
